var : Var is used to declare function scoped variable . Inside the function we can access it anywhere in the function if we change the value of the variable it make change in every part on that function which can lead to unexpected behaviour .

let : let is used to declare block scoped variable . Inside the function we only can access it where it was declared . It does not allow redelaration within same scope .

const : const is also used to declare block scoped variable . But it is only used for constant value assignment . It does not allows reassignment but allows modification of the variable's value .

Null : Null is used to explicitly represent the absence of any object value . Null can be assined to object properties to explicitly remove their values. Null can be used in comparisons to check if a variable has been explicitly set to null.

Undefined : Variables that are declared but not assigned with a value results 'undefined' . Accessing non existing object properties result in 'undefined' . A Function without an explicit return 'undefined'

REST API means Representational State Transfer Application Programming Interface . It is an architectural style for designing networked applications . It is an approach to building web services that are scalable , stateless and can be easily consumed by client . REST APIs are widely used for building web services due to their simplicity, scalability and easy integration with different techonologies . They are commonly implemented in web and mobile application development for data exchange and interaction with server side resources .
